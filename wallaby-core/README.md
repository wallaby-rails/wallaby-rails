Wallaby::Core holds all the core interfaces that [Wallaby](https://github.com/wallaby-rails/wallaby) gem is built upon.

## Install

Add `Wallaby::Core` to `Gemfile`.

```ruby
gem 'wallaby-core'
```

And re-bundle.

```shell
bundle install
```
