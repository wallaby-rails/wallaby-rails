# Wallaby Cop

Rubocop configuration for Wallaby projects.

## Installation

Put this in your `Gemfile`

```ruby
gem 'wallaby-cop'
```

## Usage

Put this into your `.rubocop.yml`.

```yaml
inherit_gem:
  wallaby-cop: rubocop.yml
```
